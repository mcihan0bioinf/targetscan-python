# Notice

This is a Python port of the original TargetScan target-prediction
toolkit. See https://www.targetscan.org for the original source, data,
and terms of use of the algorithm and any associated data files.

The algorithm, scoring model coefficients (Agarwal et al. 2015, Garcia et
al.), and several implementation quirks are reproduced from the original
for output compatibility; see code comments in `targetscan/` for specific
notes on preserved edge-case behaviors.
